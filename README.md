## My Projects

### 1. Calculator
A simple calculator that can:
- Add, subtract, multiply, divide
- Calculate remainder (%)
- Calculate power (**)
- Handle division by zero

### 2. Student Grade Tracker
A system to:
- Add students with grades in multiple subjects
- Calculate individual student averages
- Calculate class averages per subject
- Find top students
- Identify struggling students (below 70%)
- Generate report cards with letter grades

## Files
- `index.html` - The webpage
- `main.js` - All my practice exercises from Lessons 7 and 8
- `calculator.js` - The calculator project
- `grade-tracker.js` - The grade tracker project

## How to Use
1. Open `index.html` in your browser
2. Open the browser console (F12)
3. All code runs automatically
4. Click buttons to test different parts
